# streamAula — recursos web embebidos

Esta carpeta contiene la interfaz servida por el plugin (`index.html`, `styles.css`, `app.js`). Los archivos se referencian en `streamAula.jucer` como **Binary Resources** y se compilan en `BinaryData` para servirlos por HTTP desde el ejecutable.

- **Autor:** José Assiz Alcaraz Baxter  
- **Licencia de uso:** usarse libremente referenciando al autor.

No es necesario desplegar estos archivos por separado en el uso normal del plugin; el navegador los obtiene del servidor embebido.
