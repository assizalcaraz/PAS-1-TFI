Sobre este trabajo

streamAula es un plugin de audio (JUCE) pensado como trabajo final integrador de posgrado: expone un servidor HTTP embebido, lee un buffer compartido con el motor de audio del host y retransmite PCM a uno o varios clientes web. La interfaz que ves aquí consume el endpoint de streaming y reproduce el audio con la Web Audio API.
